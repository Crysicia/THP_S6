# THP-AJAX !

## :wrench: Installation guide
To install and test our awesome project :

    $ git clone https://github.com/Crysicia/THP-Ajax-Day38.git
    $ cd THP-Ajax-Day38
    $ bundle install
    $ rails db:create db:migrate db:seed
  Run it like this :

    $ rails s
    
  Then you can go to :

    $ localhost:3000
