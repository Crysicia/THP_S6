# THP-Movies !

## :wrench: Installation guide
To install and test our awesome project :

    $ git clone https://github.com/ssimonlp/THP-rails-crypto.git
    $ cd THP-rails-crypto.
    $ bundle install
    $ rails db:create
    $ rails db:migrate
  Run it like this :

    $ rails s
    
  Then you can go to :

    $ localhost:3000
