# THP landing !

## :wrench: Installation guide
To install and test our awesome project :

    $ git clone https://github.com/Crysicia/THP-CSS-Day31.git
    $ cd THP-CSS-Day31
    $ bundle install
  Run it like this :

    $ rails s
    
  Then you can go to :

    $ localhost:3000
